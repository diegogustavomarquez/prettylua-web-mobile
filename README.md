# prettylua-web-mobile

## Para instalar localmente: 

- npm install -g @ionic/cli 

- npm install 

## abrir consola y ejecutar el siguientes comando: 

- ionic serve


## Para deployar ejecutar los siguientes comandos en firebase: 

- npm install

- ionic build --prod

- firebase login

- firebase init

- firebase deploy
